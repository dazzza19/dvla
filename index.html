DVLA API Setup

To use the vehicle lookup feature, you'll need a DVLA Vehicle Enquiry API key.

Get your API key from: DVLA Developer Portal

Environment Variables (For GitHub/Netlify)

If deploying to Netlify or similar platforms, set these environment variables:

  • DVLA_API_KEY - Your DVLA API key
  • DVLA_API_URL - https://driver-vehicle-licensing.api.gov.uk

The app will automatically detect and use environment variables if available.
